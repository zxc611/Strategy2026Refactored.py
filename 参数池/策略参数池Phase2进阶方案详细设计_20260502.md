## 方案1: RSM 响应面法 (Response Surface Methodology) ⭐ 首选推荐

### 1.1 方案概述

RSM 是工业界 DOE（实验设计）的黄金标准。通过两阶段设计（筛选+优化），用最少实验量精确定位最优参数组合，同时量化模型拟合优度和预测区间。

**适用条件**:
- Phase 1 已确认关键因子≤6个 ✅
- 因子可以连续化（或离散水平≥3） ✅
- 能运行 20-30 组参数 ✅
- 需要统计显著性结论（P值、R²） ✅

**总实验量**: 8 (Plackett-Burman) + 15~17 (CCD) + 3~5 (验证) = **26~30组**
**预计耗时**: 各组平均20分钟 → **约8.5-10小时**

## 方案2: 贝叶斯优化 (Bayesian Optimization via Optuna)

### 2.1 方案概述

贝叶斯优化适用于**参数空间连续、每组运行成本高、无法预设精确设计矩阵**的场景。它利用已运行的实验结果构建代理模型（高斯过程/TPE），智能选择下一组最有希望改善目标函数的参数。

**适用条件**:
- 参数以连续值为主（离散值通过分类分布处理）✅
- 单组运行时间 > 15分钟（"昂贵黑箱函数"）✅
- 不需要显式统计显著性（不需要P值/R²）— 优化重于推断
- 可接受 30-50 组迭代 ✅

**总实验量**: 30-50 组（含10组随机初始探索）
**预计耗时**: 各组平均20分钟 → **约10-17小时**

        # subprocess.run([...], timeout=duration_seconds)

        time.sleep(duration_seconds)
        
        # 解析日志获取性能指标
        metrics = self._parse_performance_metrics()
        return self._compute_objective(metrics)
    
    def _parse_performance_metrics(self) -> Dict[str, float]:
        """从策略日志中提取性能指标"""
        metrics = {
            'total_signals': 0,
            'total_trades': 0,
            'win_rate': 0.0,
            'profit_loss_ratio': 0.0,
            'sharpe_ratio': 0.0,
            'max_drawdown': 0.0,
            'total_pnl': 0.0,
            'avg_hold_minutes': 0.0,
        }
        
        if not os.path.exists(self.log_path):
            return metrics
        
        with open(self.log_path, 'r', encoding='utf-8') as f:
            log_content = f.read()
        
        # 示例正则（根据实际日志格式调整）
        patterns = {
            'total_signals': r'total_signals[:\s=]+(\d+)',
            'total_trades': r'total_trades[:\s=]+(\d+)',
            'win_rate': r'win_rate[:\s=]+([\d.]+)',
            'sharpe_ratio': r'sharpe[_\s]*ratio[:\s=]+([\d.]+)',
            'max_drawdown': r'max[_\s]*drawdown[:\s=]+([\d.]+)',
            'total_pnl': r'total[_\s]*pnl[:\s=]+([\d.-]+)',
        }
        
        for key, pattern in patterns.items():
            match = re.search(pattern, log_content, re.IGNORECASE)
            if match:
                try:
                    metrics[key] = float(match.group(1))
                except ValueError:
                    pass
        
        # 从 analytics_service / diagnosis 输出中提取
        # 如: [Analytics] WinRate=0.45, PL_Ratio=1.8, Sharpe=1.2
        
        return metrics
    
    def _compute_objective(self, metrics: Dict[str, float]) -> float:
        """
        计算目标函数（综合评分）
        
        策略A: 单一指标最大化
        策略B: 加权组合（多目标）
        
        推荐: 夏普比率作为主目标，但也惩罚过度交易和无交易
        """
        sharpe = metrics.get('sharpe_ratio', 0.0)
        trades = metrics.get('total_trades', 0)
        dd = metrics.get('max_drawdown', 0.0)
        win_rate = metrics.get('win_rate', 0.0)
        
        if trades == 0:
            return -1000.0  # 无交易 = 极差
        
        # === 目标函数选项 ===
        
        # 选项1: 纯夏普比率
        objective = sharpe
        
        # 选项2: 经折损调整的夏普
        # objective = sharpe * (1 - dd)  # 惩罚高回撤
        
        # 选项3: Calmar比率（收益率/最大回撤）
        # objective = (metrics['total_pnl'] / max(abs(dd), 0.001))
        
        # 选项4: 综合评分（推荐）
        # objective = (
        #     0.4 * min(sharpe / 3.0, 1.0) +
        #     0.3 * win_rate +
        #     0.2 * min(trades / 20, 1.0) +
        #     0.1 * (1 - min(dd / 0.5, 1.0))
        # )
        
        return objective

def objective(trial: optuna.Trial) -> float:
    """
    Optuna 目标函数
    trial 提供参数空间，返回值越大越好
    """
    # 1. 采样参数
    search_params = define_search_space(trial)
    full_params = {**FIXED_PARAMS, **search_params}
    
    # 2. 记录参数到trial
    for key, value in search_params.items():
        trial.set_user_attr(key, value)
    
    # 3. 写入参数覆盖文件
    runner.write_override(full_params)
    
    # 4. 运行策略
    score = runner.run_strategy(duration_seconds=900)  # 15分钟
    
    # 5. 中间报告（支持剪枝）
    trial.report(score, step=1)
    if trial.should_prune():
        raise optuna.TrialPruned()
    
    return score

class EarlyStoppingCallback:
    """如果连续N组无改善，提前停止"""
    def __init__(self, patience: int = 10, min_delta: float = 0.01):
        self.patience = patience
        self.min_delta = min_delta
        self.best_score = -float('inf')
        self.no_improve_count = 0
    
    def __call__(self, study: optuna.Study, trial: optuna.Trial):
        if trial.value is None:
            return
        
        if trial.value > self.best_score + self.min_delta:
            self.best_score = trial.value
            self.no_improve_count = 0
        else:
            self.no_improve_count += 1
        
        if self.no_improve_count >= self.patience:
            study.stop()

## 方案3: 拉丁超立方采样 (Latin Hypercube Sampling, LHS)

### 3.1 方案概述

LHS 是一种**空间填充设计**（space-filling design），在给定维度的超立方内均匀采样，每一维的每一分层恰好有一个采样点。适合高维参数空间的均匀探测，不强调正交性但强调覆盖均匀性。

**适用条件**:
- 参数维度多（6-10维）✅
- 需要全面探索参数空间而非只找最优点 ✅
- 准备做后续的灵敏度分析（Sobol Indices）✅
- 可接受 36-64 组的中等规模运行 ✅

**总实验量**: 36 组（6因子×6分层）或 64 组（8因子×8分层）
**预计耗时**: 各组平均20分钟 → **约12-21小时**

### 3.2 LHS 采样与实验设计

#### 3.2.1 参数定义

| 因子 | 类型 | 下界 | 上界 | 分层数(36组) | 分层数(64组) |
|------|------|------|------|------------|------------|
| `option_width_min_threshold` | 连续 | 1.0 | 8.0 | 6 | 8 |
| `signal_cooldown_sec` | 整数 | 0 | 300 | 6 | 8 |
| `close_take_profit_ratio` | 连续 | 1.05 | 4.0 | 6 | 8 |
| `close_max_hold_days` | 整数 | 1 | 14 | 6 | 8 |
| `max_kline` | 整数(log) | 50 | 1000 | 6 | 8 |
| `position_limit_max_ratio` | 连续 | 0.05 | 0.5 | 6 | 8 |
| `close_max_chase_attempts` | 整数 | 1 | 8 | — | 8 |
| `lots_min` | 整数 | 1 | 20 | — | 8 |

        d=n_factors,
        seed=seed,
        optimization='random-cd',  # 中心化差异优化，改善均匀性
    )
    
    # 生成 (0, 1) 区间均匀分布样本
    unit_samples = sampler.random(n=n_samples)
    
    # 转换到各因子的实际范围
    factor_names = list(factor_defs.keys())
    real_samples = np.zeros_like(unit_samples)
    
    for i, name in enumerate(factor_names):
        low, high = factor_defs[name]['bounds']
        ftype = factor_defs[name]['type']
        
        if ftype == 'integer_log':
            # 对数尺度：在log空间中均匀 → 实际空间中指数分布
            log_low = np.log10(low)
            log_high = np.log10(high)
            log_vals = log_low + unit_samples[:, i] * (log_high - log_low)
            real_vals = np.power(10, log_vals)
            real_samples[:, i] = np.round(real_vals).astype(int)
        
        elif ftype == 'integer':
            # 线性整数：在 [low, high+1) 区间均匀 → 取整
            real_vals = low + unit_samples[:, i] * (high + 1 - low)
            real_samples[:, i] = np.floor(real_vals).astype(int)
        
        else:  # continuous
            real_samples[:, i] = low + unit_samples[:, i] * (high - low)
    
    # 构建 DataFrame
    df = pd.DataFrame(real_samples, columns=factor_names)
    
    # 后处理：确保整数因子在范围内
    for name in factor_names:
        if factor_defs[name]['type'] in ('integer', 'integer_log'):
            low, high = factor_defs[name]['bounds']
            df[name] = df[name].clip(low, high).astype(int)
    
    return df

def evaluate_lhs_quality(
    df: pd.DataFrame,
    factor_defs: Dict[str, Dict[str, Any]],
    output_dir: str = OUTPUT_DIR,
) -> Dict[str, float]:
    """
    评估 LHS 采样质量：
    1. 每维分层的均匀性
    2. 两两因子投影的均匀性
    3. 最小距离分析（maximin criterion）
    """
    n = len(df)
    d = len(df.columns)
    
    # 1. 分层均匀性：每维等分N层，每层应有恰好1个样本
    uniformity_scores = {}
    for name in df.columns:
        # 将每列按n分层
        bins = np.linspace(df[name].min(), df[name].max(), n + 1)
        bin_counts = np.histogram(df[name], bins=bins)[0]
        # 理想: 每层恰好1个
        uniformity_scores[name] = np.std(bin_counts)  # 标准差越小越均匀
    
    # 2. 两两投影均匀性：将2D空间划分为 √n × √n 格子
    grid_size = max(2, int(np.sqrt(n)))
    projection_scores = {}
    for i, name1 in enumerate(df.columns):
        for name2 in list(df.columns)[i+1:]:
            bins1 = np.linspace(df[name1].min(), df[name1].max(), grid_size + 1)
            bins2 = np.linspace(df[name2].min(), df[name2].max(), grid_size + 1)
            h2d, _, _ = np.histogram2d(df[name1], df[name2], bins=[bins1, bins2])
            # 每格平均样本数 = N / grid_size²
            expected = n / (grid_size * grid_size)
            projection_scores[f"{name1}×{name2}"] = np.std(h2d.flatten()) / expected
    
    # 3. Maximin 准则：最小样本间距（越大越好）
    from scipy.spatial.distance import pdist
    normalized = (df - df.min()) / (df.max() - df.min())
    distances = pdist(normalized.values)
    maximin = distances.min()
    
    quality = {
        'mean_uniformity_std': np.mean(list(uniformity_scores.values())),
        'mean_projection_std': np.mean(list(projection_scores.values())),
        'maximin_distance': maximin,
        'uniformity_by_factor': uniformity_scores,
    }
    
    # 输出报告
    logging.info("[LHS Quality] 采样质量评估:")
    logging.info(f"  一维均匀性标准差均值: {quality['mean_uniformity_std']:.4f} (越小越好)")
    logging.info(f"  二维投影均匀性均值: {quality['mean_projection_std']:.4f} (越小越好)")
    logging.info(f"  Maximin距离: {quality['maximin_distance']:.4f} (越大越好)")
    
    # 保存质量报告
    with open(os.path.join(output_dir, 'lhs_quality_report.json'), 'w', encoding='utf-8') as f:
        json.dump(quality, f, indent=2, ensure_ascii=False)
    
    return quality

def plot_lhs_diagnostics(

def sobol_sensitivity_analysis(
    df: pd.DataFrame,
    Y: np.ndarray,  # 目标函数值向量
    output_dir: str = OUTPUT_DIR,
) -> Dict[str, Dict[str, float]]:
    """
    Sobol 灵敏度分析（依赖 SALib）
    pip install SALib
    
    输入 LHS 样本 + 对应的目标函数值
    输出每个因子的一阶和总阶 Sobol 指数
    """
    try:
        from SALib.analyze import sobol
        from SALib.sample import saltelli
    except ImportError:
        logging.warning("[LHS] SALib not installed, skip Sobol analysis")
        return {}
    
    factor_names = list(df.columns)
    
    # 定义问题
    problem = {
        'num_vars': len(factor_names),
        'names': factor_names,
        'bounds': [[FACTOR_DEFS[n]['bounds'][0], FACTOR_DEFS[n]['bounds'][1]]
                   for n in factor_names],
    }
    
    # Sobol 需要 (N × (2D+2)) 样本 → 通常 N=1024, D=维度
    # 注意：LHS 36组不足以做精度 Sobol → 建议额外生成 1024×(2D+2) 样本
    # 此处提供框架，实际需要更多样本
    
    Si = sobol.analyze(problem, Y, print_to_console=False)
    
    sensitivity = {}
    for i, name in enumerate(factor_names):
        sensitivity[name] = {
            'S1': float(Si['S1'][i]),       # 一阶指数（单独贡献）
            'ST': float(Si['ST'][i]),       # 总阶指数（含交互）
            'S1_conf': float(Si['S1_conf'][i]),
        }
    
    # 排序输出
    sorted_sens = sorted(sensitivity.items(), key=lambda x: x[1]['ST'], reverse=True)
    for name, vals in sorted_sens:
        logging.info(f"  {name}: S1={vals['S1']:.4f}, ST={vals['ST']:.4f}")
    
    # 保存
    with open(os.path.join(output_dir, 'sobol_sensitivity.json'), 'w', encoding='utf-8') as f:
        json.dump(sensitivity, f, indent=2, ensure_ascii=False)
    
    return sensitivity

"""
LHS 结果分析 — 从所有组运行完毕后执行

def analyze_lhs_results(
    df_samples: pd.DataFrame,  # N×D 参数矩阵
    results_dict: Dict[int, Dict[str, float]],  # {trial_id: {metric: value}}
    output_dir: str = OUTPUT_DIR,
):
    """
    分析 LHS 全部实验结果
    """
    # 合并参数和结果
    Y_vec = np.array([results_dict[i]['sharpe_ratio'] for i in range(len(df_samples))])
    
    # === 1. 相关性热图 ===
    corr_df = pd.concat([
        df_samples,
        pd.DataFrame({'sharpe': Y_vec})
    ], axis=1)
    corr_matrix = corr_df.corr()
    
    fig, ax = plt.subplots(figsize=(10, 8))
    im = ax.imshow(corr_matrix, cmap='RdBu_r', vmin=-1, vmax=1)
    plt.colorbar(im)
    # ... label axes
    plt.savefig(os.path.join(output_dir, 'lhs_correlation_heatmap.png'))
    
    # === 2. 识别 top 参数组合 ===
    top_n = max(3, len(Y_vec) // 10)
    top_indices = np.argsort(Y_vec)[-top_n:][::-1]
    
    logging.info(f"[LHS Analysis] Top {top_n} 组参数:")
    for rank, idx in enumerate(top_indices):
        row = df_samples.iloc[idx]
        score = Y_vec[idx]
        logging.info(f"  #{rank+1}: score={score:.4f}")
        for name in df_samples.columns:
            logging.info(f"    {name} = {row[name]}")
    
    # === 3. 平行坐标图 ===
    from pandas.plotting import parallel_coordinates
    df_plot = df_samples.copy()
    df_plot['score'] = Y_vec
    df_plot['score_bin'] = pd.cut(df_plot['score'], bins=5)
    
    fig, ax = plt.subplots(figsize=(14, 6))
    parallel_coordinates(df_plot, 'score_bin', colormap='viridis')
    plt.savefig(os.path.join(output_dir, 'lhs_parallel_coordinates.png'))
    
    # === 4. 条件推断（哪些参数区间产生高分） ===
    high_score_mask = Y_vec >= np.percentile(Y_vec, 75)
    low_score_mask = Y_vec <= np.percentile(Y_vec, 25)
    
    inference = {}
    for name in df_samples.columns:
        high_mean = df_samples.loc[high_score_mask, name].mean()
        low_mean = df_samples.loc[low_score_mask, name].mean()
        inference[name] = {
            'high_percentile_mean': high_mean,
            'low_percentile_mean': low_mean,
            'delta': high_mean - low_mean,
        }
    
    logging.info("[LHS Analysis] 高分vs低分 参数差异:")
    sorted_inf = sorted(inference.items(), key=lambda x: abs(x[1]['delta']), reverse=True)
    for name, vals in sorted_inf:
        logging.info(f"  {name}: high={vals['high_percentile_mean']:.3f}, "
                    f"low={vals['low_percentile_mean']:.3f}, Δ={vals['delta']:.3f}")
