### SER-01：datetime序列化链路

```
定义端：serialization_utils.py
  └─ json_default_serializer: datetime → {"__datetime__": iso_string}
  └─ json_object_hook: {"__datetime__": str} → datetime

传递端：order_service.py/position_service.py等19个文件
  └─ json_dumps(data) → json.dumps(data, default=json_default_serializer)
  └─ json_loads(json_str) → json.loads(json_str, object_hook=json_object_hook)

消费端：数据持久化与读取
  └─ 写入：datetime → JSON字符串（带特殊标记）
  └─ 读取：JSON字符串 → datetime（还原）

结果验证：
  - datetime(2026,5,25) → 写入 → 读取 → datetime(2026,5,25) ✅
  - np.nan → 写入 → 读取 → np.nan ✅
```
