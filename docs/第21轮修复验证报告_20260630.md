### 1.1 守恒断言bug修复

**问题**: `_audit_conservation`方法中`_expired_cutoff`未定义导致NameError

**修复**:

2. 增加"注册过程中跳过检查"逻辑（`_option_info < 10000`时跳过）

**文件**: `width_cache_state_mixin.py:154-195`

### 1.2 runtime_subscribed诊断误判修复

**问题**: 日志显示`runtime_subscribed=0`，但实际订阅已完成

**根因**: `_get_runtime_state()`检查`runtime_strategy/params/runtime_core`的`_subscribed_instruments`，但实际订阅列表在`_lifecycle_platform._subscribed_instruments`中维护

**修复**: 扩展检查范围，增加对`_lifecycle_platform._subscribed_instruments`的检查

**文件**: `health_monitor.py:2228-2234`

**修改前**:
```python
_raw_subscribed = None
for _src in [runtime_strategy, getattr(runtime_strategy, 'params', None), runtime_core]:
    if _src is not None:
        _raw_subscribed = getattr(_src, '_subscribed_instruments', None)
        if _raw_subscribed:
            break
runtime_subscribed = set(_raw_subscribed) if _raw_subscribed else set()
```

**修改后**:
```python
_raw_subscribed = None
for _src in [runtime_strategy, getattr(runtime_strategy, 'params', None), runtime_core]:
    if _src is not None:
        _raw_subscribed = getattr(_src, '_subscribed_instruments', None)
        if _raw_subscribed:
            break
# 修复：检查_lifecycle_platform（实际订阅列表维护在此处）
if not _raw_subscribed and runtime_core is not None:
    _lp = getattr(runtime_core, '_lifecycle_platform', None)
    if _lp is not None:
        _raw_subscribed = getattr(_lp, '_subscribed_instruments', None)
runtime_subscribed = set(_raw_subscribed) if _raw_subscribed else set()
```
