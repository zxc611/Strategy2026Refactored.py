#### 2.1.2 亏损模式根因

89% 的交易亏损**恰好 -1.00 点**，这是期权买卖价差的典型特征：

```
开仓价(Ask) = 平仓价(Bid) + 1个最小变动价位
→ 每笔交易必然亏损至少1个最小变动价位
```

策略本质上是**每 5 分钟系统性支付买卖价差成本的“价差成本生成器”**，不具备任何正向期望。

|------|-----|------------|---------------|
| 止盈(tp_ratio=1.8) | sp=open×1.8 | **+80%** | ≈0% |
| 止损(sl_ratio=0.3) | sl=open×0.7 | **-30%** | <1% |
