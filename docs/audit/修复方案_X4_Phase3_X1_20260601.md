### 1.3 data_access.py需新增的Protocol接口

当前5个Protocol覆盖了市场数据/持仓/交易/风控状态读写，**但缺少通用查询和写入能力**，需新增3个Protocol：

#### 新增Protocol 1：`DataWriteAccess`（覆盖ds_data_writer.py的connect+写入）

```python
@runtime_checkable
class DataWriteAccess(Protocol):
    def write_records(self, table: str, records: List[Dict[str, Any]],
                      strategy_id: Optional[str] = None) -> WriteResult: ...
    def flush_buffer(self, table: str) -> FlushResult: ...
```

#### 新增Protocol 2：`QueryAccess`（覆盖serialization_utils.py的connect_in_memory+fetchdf）

```python
@runtime_checkable
class QueryAccess(Protocol):
    def execute_query(self, query: str, params: Optional[List] = None,
                      in_memory: bool = False) -> QueryResult: ...
    def register_dataframe(self, name: str, df: Any) -> None: ...
```

#### 新增Protocol 3：`MaintenanceAccess`（覆盖maintenance_service.py的connect+维护操作）

```python
@runtime_checkable
class MaintenanceAccess(Protocol):
    def compact_database(self) -> CompactionResult: ...
    def rebuild_index(self, table: str) -> IndexResult: ...
    def get_storage_stats(self) -> StorageStatsResult: ...
```
