> 决策目标：评估并选择最优的期权过滤方法，平衡理论严谨性与实盘有效性

## 一、问题背景

### 1.1 核心矛盾

在[三层期权五态排序方案_最终落地方案_20260624.md]中，Layer 1的前置硬过滤采用了基于Greeks的过滤方法：

```python
def _greeks_hard_filter(delta: float, gamma: float, theta: float,
                        option_price: float, month_type: str) -> bool:
    """
    Greeks前置硬过滤。
    不满足条件的期权直接排除，不参与后续排序。
    """
    # Delta范围：排除深度虚值/实值
    if abs(delta) < 0.15 or abs(delta) > 0.85:
        return False

    # 近月Gamma约束：状态翻转过快
    if month_type in ('front_month', 'next_month') and gamma >= 0.08:
        return False

    # Theta约束：近月时间价值损耗过高
    if month_type == 'front_month' and gamma > 0.03:
        theta_pct = abs(theta) / option_price
        if theta_pct > 0.015:  # 日损耗超过1.5%直接排除
            return False

    return True
```

**问题**：该过滤方法基于期权定价理论（Greeks），与期权五态统计（CR/CF/WR/WF/Other）的自然状态描述存在差异。期权五态统计直接反映期权同步于当月期货移动的实际表现，可能更符合实盘现状。

### 1.2 决策问题

**核心问题**：应该采用哪种过滤方法？
- **方法A**：Greeks硬过滤（文档方案）
- **方法B**：期权五态统计过滤（直觉方案）
- **方法C**：融合方案（五态为主，Greeks为辅）

---

## 二、两种过滤方法的深度对比

### 2.1 方法A：Greeks硬过滤（文档方案）

#### 理论基础
- **Delta过滤**：排除深度虚值（Delta < 0.15）和深度实值（Delta > 0.85）
  - 理由：深度虚值期权流动性差、预测能力弱；深度实值期权杠杆低、信号价值低
- **Gamma过滤**：近月Gamma >= 0.08的期权排除
  - 理由：Gamma过大导致状态翻转过快，信号不稳定
- **Theta过滤**：近月时间价值损耗 > 1.5%/日的期权排除
  - 理由：时间价值损耗过快，持有成本高

#### 优势
1. **理论严谨**：基于期权定价理论，有明确的金融学基础
2. **参数可调**：Delta/Gamma/Theta阈值可根据市场环境调整
3. **计算简单**：Greeks可实时计算，无需历史数据积累
4. **风险控制**：直接排除高风险期权（高Gamma、高Theta）

#### 劣势
1. **理论假设**：基于Black-Scholes等定价模型，可能与实盘偏离
2. **忽略实盘表现**：不直接考虑期权的历史预测准确率
3. **参数主观**：阈值（0.15, 0.85, 0.08, 0.015）缺乏实证校准
4. **可能过度过滤**：可能排除实际表现良好的期权

### 2.2 方法B：期权五态统计过滤（直觉方案）

#### 理论基础
- **五态定义**：
  - **CR (Correct Rise)**：期权方向正确，期货上涨
  - **CF (Correct Fall)**：期权方向正确，期货下跌
  - **WR (Wrong Rise)**：期权方向错误，期货上涨
  - **WF (Wrong Fall)**：期权方向错误，期货下跌
  - **Other**：其他状态（平盘、数据缺失等）
- **过滤逻辑**：基于期权历史五态统计，直接反映期权的预测能力

#### 实现方案

```python
def five_state_filter(five_state_stats: dict,
                      min_total_count: int = 10,
                      min_correct_rate: float = 0.35) -> bool:
    """
    基于期权五态统计的过滤方法。

    参数：
    - five_state_stats: 期权的五态统计数据
        {
            'CR': int,  # 正确上涨次数
            'CF': int,  # 正确下跌次数
            'WR': int,  # 错误上涨次数
            'WF': int,  # 错误下跌次数
            'Other': int,  # 其他状态次数
            'Total': int,  # 总次数
        }
    - min_total_count: 最小样本数要求
    - min_correct_rate: 最小正确率阈值

    返回：
    - True: 保留该期权
    - False: 排除该期权
    """
    total_count = five_state_stats.get('Total', 0)

    # 样本不足，无法判断
    if total_count < min_total_count:
        return False  # 或返回True，取决于策略

    cr_count = five_state_stats.get('CR', 0)
    cf_count = five_state_stats.get('CF', 0)
    correct_count = cr_count + cf_count

    # 计算正确率
    correct_rate = correct_count / total_count

    # 过滤条件
    if correct_rate < min_correct_rate:
        return False

    return True
```

#### 优势
1. **实盘驱动**：直接反映期权的历史预测能力
2. **数据驱动**：基于实际交易数据，非理论假设
3. **直观易懂**：正确率是直接的绩效指标
4. **自适应**：随着数据积累，过滤效果自动优化

#### 劣势
1. **数据依赖**：需要足够的历史数据积累
2. **滞后性**：历史表现不能保证未来表现
3. **样本偏差**：新上市期权或交易稀疏的期权样本不足
4. **过拟合风险**：可能过度依赖历史数据

### 2.3 方法C：融合方案（五态为主，Greeks为辅）

#### 设计理念
- **主过滤**：基于五态统计的硬过滤（反映实盘表现）
- **辅助修正**：基于Greeks的软修正（理论风险控制）
- **样本不足兜底**：当五态样本不足时，使用Greeks过滤

#### 实现方案

```python
def hybrid_filter(delta: float, gamma: float, theta: float,
                  option_price: float, month_type: str,
                  five_state_stats: dict,
                  config: dict = None) -> tuple[bool, float]:
    """
    融合过滤方案：五态为主，Greeks为辅。

    返回：
    - (是否保留, 惩罚系数)
    """
    if config is None:
        config = {
            'min_total_count': 10,
            'min_correct_rate': 0.35,
            'delta_hard_min': 0.10,  # 更宽松的硬过滤
            'delta_hard_max': 0.90,
            'gamma_hard_threshold': 0.10,  # 更宽松
            'theta_hard_pct': 0.020,  # 更宽松
        }

    total_count = five_state_stats.get('Total', 0)

    # 情况1：样本不足，使用Greeks硬过滤
    if total_count < config['min_total_count']:
        return greeks_hard_filter_lite(delta, gamma, theta, option_price, month_type, config), 1.0

    # 情况2：样本充足，使用五态硬过滤
    cr_count = five_state_stats.get('CR', 0)
    cf_count = five_state_stats.get('CF', 0)
    correct_rate = (cr_count + cf_count) / total_count

    if correct_rate < config['min_correct_rate']:
        return False, 0.0  # 五态硬过滤排除

    # 情况3：五态过滤通过，应用Greeks软修正
    penalty = compute_greeks_penalty(delta, gamma, theta, option_price, month_type, config)

    return True, penalty


def greeks_hard_filter_lite(delta: float, gamma: float, theta: float,
                            option_price: float, month_type: str,
                            config: dict) -> bool:
    """
    简化版Greeks硬过滤（样本不足时使用）。
    阈值比文档方案更宽松。
    """
    # Delta硬过滤
    if abs(delta) < config['delta_hard_min'] or abs(delta) > config['delta_hard_max']:
        return False

    # Gamma硬过滤
    if month_type in ('front_month', 'next_month') and gamma >= config['gamma_hard_threshold']:
        return False

    # Theta硬过滤
    if month_type == 'front_month' and gamma > 0.03:
        theta_pct = abs(theta) / option_price
        if theta_pct > config['theta_hard_pct']:
            return False

    return True


def compute_greeks_penalty(delta: float, gamma: float, theta: float,
                          option_price: float, month_type: str,
                          config: dict) -> float:
    """
    计算Greeks软修正惩罚系数（0-1）。
    返回值越小，惩罚越重。
    """
    penalty = 1.0

    # Delta惩罚
    if abs(delta) < 0.15:
        penalty *= 0.85  # 深度虚值降权15%
    elif abs(delta) > 0.85:
        penalty *= 0.90  # 深度实值降权10%

    # Gamma惩罚
    if month_type in ('front_month', 'next_month'):
        if gamma >= 0.08:
            penalty *= 0.80  # 高Gamma降权20%
        elif gamma >= 0.06:
            penalty *= 0.90  # 中Gamma降权10%

    # Theta惩罚
    if month_type == 'front_month' and gamma > 0.03:
        theta_pct = abs(theta) / option_price
        if theta_pct > 0.015:
            penalty *= 0.85  # 高Theta损耗降权15%
        elif theta_pct > 0.010:
            penalty *= 0.92  # 中Theta损耗降权8%

    return penalty
```

#### 优势
1. **兼顾理论与实盘**：五态反映实盘，Greeks控制理论风险
2. **自适应**：样本充足时用五态，样本不足时用Greeks
3. **风险可控**：Greeks软修正提供额外保护层
4. **灵活性高**：可调整五态/Greeks的权重

#### 劣势
1. **复杂度增加**：需要维护两套过滤逻辑
2. **参数更多**：需要校准的参数增加
3. **计算开销**：需要同时计算五态统计和Greeks

## 三、决策框架与验证方案

### 3.1 决策流程图

```
开始
  ↓
是否有足够的历史五态数据（≥6个月）？
  ├─ 否 → 使用Greeks过滤（文档方案）
  │        ↓
  │      开始积累五态数据
  │        ↓
  │      3个月后重新评估
  │
  └─ 是 ↓
      执行对比验证（详见3.2）
        ↓
      生成验证报告
        ↓
      两种方法重叠率 > 80%？
        ├─ 是 → 两种方法等效
        │        ↓
        │      选择实现更简单的方案（Greeks）
        │
        └─ 否 ↓
            五态过滤正确率 > Greeks过滤？
              ├─ 是 → 采用融合方案（五态为主）
              │        ↓
              │      Phase 1: 五态硬过滤 + Greeks软修正
              │      Phase 2: 根据表现调整权重
              │
              └─ 否 → 采用文档方案（Greeks硬过滤）
                       ↓
                     持续监控五态数据
                       ↓
                     定期重新评估
```

### 3.2 对比验证方案

#### 验证目标
1. 两种过滤方法的重叠率
2. 各自保留期权的预测准确率
3. 各自的过滤率（排除比例）
4. 各自的Calmar比率
5. 各自的最大回撤

#### 验证脚本

```python
import pandas as pd
import numpy as np
from typing import Dict, List, Tuple

class FilterMethodValidator:
    """
    期权过滤方法对比验证器。
    """

    def __init__(self, option_data: pd.DataFrame, lookback_months: int = 6):
        """
        参数：
        - option_data: 期权历史数据，包含：
            - option_id: 期权ID
            - delta, gamma, theta, vega: Greeks
            - option_price: 期权价格
            - month_type: 月份类型（front_month, next_month, etc.）
            - CR, CF, WR, WF, Other, Total: 五态统计
            - next_day_return: 次日收益率（用于验证）
        - lookback_months: 回看月数
        """
        self.data = option_data
        self.lookback_months = lookback_months
        self.results = {}

    def validate_all_methods(self) -> Dict:
        """
        验证所有过滤方法。
        """
        print("=" * 80)
        print("期权过滤方法对比验证报告")
        print("=" * 80)
        print(f"数据范围：过去{self.lookback_months}个月")
        print(f"期权总数：{len(self.data)}")
        print()

        # 方法A：Greeks硬过滤
        result_a = self._validate_greeks_filter()
        self.results['greeks'] = result_a

        # 方法B：五态过滤
        result_b = self._validate_five_state_filter()
        self.results['five_state'] = result_b

        # 方法C：融合方案
        result_c = self._validate_hybrid_filter()
        self.results['hybrid'] = result_c

        # 对比分析
        comparison = self._compare_methods()

        return {
            'greeks': result_a,
            'five_state': result_b,
            'hybrid': result_c,
            'comparison': comparison
        }

    def _validate_greeks_filter(self) -> Dict:
        """
        验证Greeks硬过滤方法。
        """
        print("方法A：Greeks硬过滤")
        print("-" * 80)

        # 应用过滤
        filtered_mask = self.data.apply(
            lambda row: self._greeks_hard_filter(
                row['delta'], row['gamma'], row['theta'],
                row['option_price'], row['month_type']
            ),
            axis=1
        )

        filtered_data = self.data[filtered_mask]
        excluded_data = self.data[~filtered_mask]

        # 计算指标
        filter_rate = 1 - len(filtered_data) / len(self.data)
        correct_rate = (filtered_data['CR'].sum() + filtered_data['CF'].sum()) / filtered_data['Total'].sum()
        avg_return = filtered_data['next_day_return'].mean()

        result = {
            'method': 'Greeks硬过滤',
            'total_options': len(self.data),
            'retained_options': len(filtered_data),
            'excluded_options': len(excluded_data),
            'filter_rate': filter_rate,
            'correct_rate': correct_rate,
            'avg_return': avg_return,
            'retained_ids': set(filtered_data['option_id'].tolist())
        }

        self._print_result(result)
        return result

    def _validate_five_state_filter(self) -> Dict:
        """
        验证五态过滤方法。
        """
        print("\n方法B：五态统计过滤")
        print("-" * 80)

        # 应用过滤
        filtered_mask = self.data.apply(
            lambda row: self._five_state_filter(
                row['CR'], row['CF'], row['WR'], row['WF'], row['Other'], row['Total']
            ),
            axis=1
        )

        filtered_data = self.data[filtered_mask]
        excluded_data = self.data[~filtered_mask]

        # 计算指标
        filter_rate = 1 - len(filtered_data) / len(self.data)
        correct_rate = (filtered_data['CR'].sum() + filtered_data['CF'].sum()) / filtered_data['Total'].sum()
        avg_return = filtered_data['next_day_return'].mean()

        result = {
            'method': '五态统计过滤',
            'total_options': len(self.data),
            'retained_options': len(filtered_data),
            'excluded_options': len(excluded_data),
            'filter_rate': filter_rate,
            'correct_rate': correct_rate,
            'avg_return': avg_return,

    def _validate_hybrid_filter(self) -> Dict:
        """
        验证融合方案。
        """
        print("\n方法C：融合方案（五态为主，Greeks为辅）")
        print("-" * 80)

        # 应用过滤
        results = self.data.apply(
            lambda row: self._hybrid_filter(
                row['delta'], row['gamma'], row['theta'],
                row['option_price'], row['month_type'],
                row['CR'], row['CF'], row['WR'], row['WF'], row['Other'], row['Total']
            ),
            axis=1
        )

        filtered_mask = results.apply(lambda x: x[0])
        penalties = results.apply(lambda x: x[1])

        filtered_data = self.data[filtered_mask].copy()
        filtered_data['penalty'] = penalties[filtered_mask]

        # 计算指标
        filter_rate = 1 - len(filtered_data) / len(self.data)
        correct_rate = (filtered_data['CR'].sum() + filtered_data['CF'].sum()) / filtered_data['Total'].sum()
        avg_return = filtered_data['next_day_return'].mean()
        avg_penalty = filtered_data['penalty'].mean()

        result = {
            'method': '融合方案',
            'total_options': len(self.data),
            'retained_options': len(filtered_data),
            'excluded_options': len(self.data) - len(filtered_data),
            'filter_rate': filter_rate,
            'correct_rate': correct_rate,
            'avg_return': avg_return,
            'avg_penalty': avg_penalty,
            'retained_ids': set(filtered_data['option_id'].tolist())
        }

        self._print_result(result)
        return result

    def _compare_methods(self) -> Dict:
        """
        对比三种方法。

        print("\n" + "=" * 80)
        print("对比分析")
        print("=" * 80)

        greeks_ids = self.results['greeks']['retained_ids']
        five_state_ids = self.results['five_state']['retained_ids']
        hybrid_ids = self.results['hybrid']['retained_ids']

        # 计算重叠率
        overlap_greeks_five_state = len(greeks_ids & five_state_ids) / len(greeks_ids | five_state_ids)
        overlap_greeks_hybrid = len(greeks_ids & hybrid_ids) / len(greeks_ids | hybrid_ids)
        overlap_five_state_hybrid = len(five_state_ids & hybrid_ids) / len(five_state_ids | hybrid_ids)

        print(f"\n重叠率分析：")
        print(f"  Greeks vs 五态: {overlap_greeks_five_state:.2%}")
        print(f"  Greeks vs 融合: {overlap_greeks_hybrid:.2%}")
        print(f"  五态 vs 融合: {overlap_five_state_hybrid:.2%}")

        # 独有分析
        greeks_only = greeks_ids - five_state_ids
        five_state_only = five_state_ids - greeks_ids

        print(f"\n独有期权分析：")
        print(f"  仅Greeks保留: {len(greeks_only)}个")
        print(f"  仅五态保留: {len(five_state_only)}个")

        # 正确率对比
        print(f"\n正确率对比：")
        print(f"  Greeks: {self.results['greeks']['correct_rate']:.2%}")
        print(f"  五态: {self.results['five_state']['correct_rate']:.2%}")
        print(f"  融合: {self.results['hybrid']['correct_rate']:.2%}")

        # 推荐决策
        print(f"\n推荐决策：")
        if overlap_greeks_five_state > 0.80:
            print("  ✓ 两种方法等效，建议使用Greeks过滤（实现更简单）")
        elif self.results['five_state']['correct_rate'] > self.results['greeks']['correct_rate'] + 0.05:
            print("  ✓ 五态过滤显著优于Greeks过滤，建议采用融合方案")
        elif self.results['greeks']['correct_rate'] > self.results['five_state']['correct_rate'] + 0.05:
            print("  ✓ Greeks过滤显著优于五态过滤，建议采用文档方案")
        else:
            print("  ✓ 两种方法差异不显著，建议采用融合方案（兼顾理论与实盘）")

        return {
            'overlap_greeks_five_state': overlap_greeks_five_state,
            'overlap_greeks_hybrid': overlap_greeks_hybrid,
            'overlap_five_state_hybrid': overlap_five_state_hybrid,
            'greeks_only_count': len(greeks_only),
            'five_state_only_count': len(five_state_only)
        }

    def _greeks_hard_filter(self, delta: float, gamma: float, theta: float,
                           option_price: float, month_type: str) -> bool:
        """
        Greeks硬过滤（文档方案）。
        """
        # Delta范围
        if abs(delta) < 0.15 or abs(delta) > 0.85:
            return False

        # Gamma约束
        if month_type in ('front_month', 'next_month') and gamma >= 0.08:
            return False

        # Theta约束
        if month_type == 'front_month' and gamma > 0.03:
            theta_pct = abs(theta) / option_price
            if theta_pct > 0.015:
                return False

        return True

    def _five_state_filter(self, cr: int, cf: int, wr: int, wf: int,
                          other: int, total: int) -> bool:
        """
        五态过滤。
        """
        if total < 10:
            return False

        correct_rate = (cr + cf) / total
        return correct_rate >= 0.35

    def _hybrid_filter(self, delta: float, gamma: float, theta: float,
                      option_price: float, month_type: str,
                      cr: int, cf: int, wr: int, wf: int,
                      other: int, total: int) -> Tuple[bool, float]:
        """
        融合过滤。
        """
        # 样本不足，使用Greeks硬过滤
        if total < 10:
            return self._greeks_hard_filter(delta, gamma, theta, option_price, month_type), 1.0

        # 五态硬过滤
        correct_rate = (cr + cf) / total
        if correct_rate < 0.35:
            return False, 0.0

        # Greeks软修正
        penalty = self._compute_greeks_penalty(delta, gamma, theta, option_price, month_type)

        return True, penalty

    def _compute_greeks_penalty(self, delta: float, gamma: float, theta: float,
                               option_price: float, month_type: str) -> float:
        """
        计算Greeks惩罚系数。
        """
        penalty = 1.0

        if abs(delta) < 0.15:
            penalty *= 0.85
        elif abs(delta) > 0.85:
            penalty *= 0.90

        if month_type in ('front_month', 'next_month'):
            if gamma >= 0.08:
                penalty *= 0.80
            elif gamma >= 0.06:
                penalty *= 0.90

        if month_type == 'front_month' and gamma > 0.03:
            theta_pct = abs(theta) / option_price
            if theta_pct > 0.015:
                penalty *= 0.85
            elif theta_pct > 0.010:
                penalty *= 0.92

        return penalty

    def _print_result(self, result: Dict):
        """
        打印结果。
        """
        print(f"  保留期权数: {result['retained_options']} / {result['total_options']}")
        print(f"  过滤率: {result['filter_rate']:.2%}")
        print(f"  正确率: {result['correct_rate']:.2%}")
        print(f"  平均收益: {result['avg_return']:.4f}")
        if 'avg_penalty' in result:
            print(f"  平均惩罚系数: {result['avg_penalty']:.4f}")

    def generate_report(self, output_path: str):
        """
        生成详细报告。
        """
        report = f"""
```

## 七、最终推荐方案

### 7.1 推荐方案

**推荐采用：融合方案（五态为主，Greeks为辅）**

**理由**：
1. **兼顾理论与实盘**：五态统计反映实盘表现，Greeks提供理论风险控制
2. **自适应性强**：样本充足时用五态，样本不足时用Greeks
3. **风险可控**：Greeks软修正提供额外保护层
4. **灵活性高**：可根据实盘表现调整五态/Greeks的权重
