- risk_amount = 100000 * 0.02 = 2000
- stop_distance = 2
- size = 2000 / 2 = 1000

**核查结果**: 使用固定风险法，受single_position_cap限制 ✅

### 5.2 large模式 — 凯利公式

- 输入: equity=100000, entry=100, stop=98, win_rate=0.6, win_loss_ratio=2.0
- kelly_fraction=0.33，条件 `0.33 > 0` 为 True，走凯利公式
- kelly_fraction(0.6, 2.0, 0.33) = 0.132
- risk_amount = 100000 * 0.132 = 13200
- stop_distance = 2
- shares = 13200 / 2 = 6600
- max_shares = 100000 * 0.08 / 100 = 80
- 最终结果: min(6600, 80) = **80.00**

**核查结果**: 使用凯利公式，受single_position_cap限制 ✅

### 5.3 方法差异确认

- small模式结果: 400.00 (固定风险法)
- large模式结果: 80.00 (凯利公式)
- 两种方法结果不同，无混用 ✅
