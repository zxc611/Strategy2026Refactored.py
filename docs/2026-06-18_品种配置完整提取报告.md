## 四、配置文件格式

### 4.1 期货配置格式

```
交易所.合约代码|internal_id|产品代码|年月
```

**示例**:
```
CFFEX.IC2609|1|IC|2609
CZCE.CF607|19|CF|2607
SHFE.cu2607|25|cu|2607
```

### 4.2 期权配置格式

```
交易所.期权代码|internal_id|underlying_future_id|期权产品代码||年月|类型|行权价
```

**示例**:
```
CZCE.AP610C10000|1|0|AP_O||2608|CALL|10000.00000000
SHFE.au2607C616|100|0|au_O||2607|CALL|616.00000000
```

**注意**: underlying_future_id字段为0，需要后续填充。
