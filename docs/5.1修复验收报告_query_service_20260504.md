| 项目 | 详情 |

### 2. [P1] QueryService直接访问`storage._params_service`私有属性(6处)

| 项目 | 详情 |
|------|------|
| **问题** | 6处穿透`self._storage._params_service`访问私有属性，违反封装，与storage实现耦合 |
| **修复** | 新增`_get_params_service()`方法，通过`get_params_service()`公共单例获取，6处全部替换 |
| **修改点** | L296 `_get_cached_instruments`, L355-356 `get_exchange`, L409 `get_registered_instruments`, L420 `get_registered_instruments`, L1038 `get_all_instrument_summaries` |

| **验收** | **PASS** - `_storage._params_service`完全消除(0处残留) |

### 3. [P1] `_internal_id_to_instrument_idx`缓存无锁保护

| 项目 | 详情 |
|------|------|
| **问题** | `_ensure_internal_id_index`中`_internal_id_to_instrument_idx`字典构建无锁，多线程可能读到半构建状态 |
| **修复** | (1) 新增 `self._idx_lock = threading.Lock()` (2) `_ensure_internal_id_index`整体包裹在 `with self._idx_lock:` 内 |
| **功能影响** | 无。锁仅保证构建原子性，构建完成后直接返回字典引用(读操作无锁，Python GIL保证引用安全) |
| **验收** | **PASS** |
