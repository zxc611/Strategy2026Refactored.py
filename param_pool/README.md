## V8.0核心变更

1. **五态真实化**：移植`width_cache._classify_status()`判定表
2. **Greeks真实化**：BS解析解+IV牛顿迭代法
3. **动态滑点**：按`bid_ask_spread`实时调整
4. **逻辑反转平仓**：信号反转时立即离场
5. **P0绿灯检验**：8项硬编码检验+三级判决

## P0绿灯检验项

| # | 检验项 | 阈值 | 性质 |
|---|--------|------|------|
| 1 | 样本外衰减 | ≥-30% | 不可妥协 |
| 2 | 训练夏普 | ≥0.5 | 不可妥协 |
| 3 | 样本外夏普 | ≥0.3 | 不可妥协 |
| 4 | 最大回撤 | ≥-50% | 不可妥协 |
| 5 | 信号数 | ≥30 | 不可妥协 |
| 6 | 逻辑反转阈值 | ≥0.8 | 不可妥协 |
| 7 | 风险收益比 | 止盈>止损 | 不可妥协 |
| 8 | intuition参数 | 无 | 不可妥协铁律 |

## 运行流程

```bash
# 1. 数据预处理
python 参数池/preprocess_ticks.py

# 2. 两轮回测 + P0绿灯检验
python 参数池/task_scheduler.py

# 3. 参数敏感性分析（可选）
python 参数池/sensitivity_analysis.py
```
