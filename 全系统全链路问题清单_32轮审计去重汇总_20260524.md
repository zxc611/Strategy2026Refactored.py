**问题指纹码生成规则**:
```
指纹码 = MD5(文件路径 + 问题类型 + 问题描述关键词前50字符)
```

**去重判定规则**:
1. 同一文件同一问题的不同轮次发现 → 保留首次发现，标记交叉确认次数
2. 同一问题不同表述 → 合并为一条，保留最详细描述
3. 子问题与父问题 → 保留父问题，子问题作为补充说明
